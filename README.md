# 🍱 방산고 급식 알리미

> 방산고등학교 학생들을 위한 급식 정보 웹 서비스

`HTML5` `CSS3` `JavaScript` `Render` `NEIS Open API`

---

## 📌 프로젝트 소개

매일 학교 급식 메뉴를 확인하기 불편하다는 점에서 출발했습니다.  
교육부 NEIS 공공 API를 활용해 방산고등학교의 당일 중식 메뉴를 간편하게 확인할 수 있는 웹 서비스입니다.

🔗 **배포 주소**: `bangsan.onrender.com`

---

## ✨ 주요 기능

- **당일 급식 자동 표시** — 오후 2시 이후 접속 시 다음 급식일로 자동 전환
- **날짜 네비게이션** — 전날/다음날 화살표 버튼으로 최대 ±5일 조회
- **주말·공휴일 자동 스킵** — 평일 기준으로만 날짜 이동
- **스와이프 제스처** — 모바일에서 좌우 스와이프로 날짜 이동
- **스켈레톤 로딩** — 데이터 로딩 중 뼈대 UI 표시
- **페이지 넘김 애니메이션** — 날짜 이동 시 슬라이드 전환 효과
- **탭 타이틀 동기화** — 현재 조회 날짜가 브라우저 탭에 표시
- **학사일정 바로가기** — 하단 버튼으로 학사일정 페이지로 이동
- **알레르기 번호 자동 제거** — 메뉴명에 붙는 알레르기 코드 파싱 후 제거
- **초기화 버튼** — 날짜 이동 후 기준일로 빠르게 복귀

## 🔧 트러블슈팅

**1. 알레르기 번호 파싱 문제**  
NEIS API 응답에서 메뉴명에 `비빔밥(달걀)/고추장5` 형태로 알레르기 번호가 붙어 나오는 문제 발견.  
괄호 안 숫자뿐 아니라 메뉴명에 직접 붙은 숫자도 정규식으로 제거하는 방식으로 해결.

```js
raw.replace(/\([^)]*\)/g, '')      // 괄호 안 숫자 제거
   .replace(/[\d.]+(?=\s*(<br\/>|$))/g, '') // 줄 끝 숫자 제거
   .replace(/\d+(\.\d+)*/g, '')    // 나머지 붙어있는 숫자 제거
```

**2. Render 배포 방식 설정 오류**  
처음 배포 시 서비스 타입을 잘못 설정해 `failed to read dockerfile` 오류 발생.  
단순 정적 파일 서빙임을 파악하고 **Static Site** 타입으로 변경, Publish Directory를 `.`으로 설정해 해결.

**3. 시간대(Timezone) 문제**  
Render 서버가 UTC 기준이라 한국 시간(KST)과 9시간 차이 우려.  
날짜 계산 로직이 서버가 아닌 **클라이언트 브라우저**에서 실행되므로 사용자의 로컬 시간이 그대로 적용됨을 확인, 별도 처리 불필요.
